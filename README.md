#  Event Reminder

A simple web-based application to help you **remember important events** with ease.  
Built using **HTML, CSS, and JavaScript**, this project is beginner-friendly and demonstrates how to manage reminders in a clean and interactive way.

---

##  Features
- Add custom events with a title and date.
- Display upcoming events in a neat list.
- Simple and responsive design.
- Lightweight and easy to run in any browser.

---

##  Tech Stack
- **HTML** – Structure of the app
- **CSS** – Styling and layout
- **JavaScript** – Event logic and interactivity
